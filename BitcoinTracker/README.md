# Bitcoin Tracker

The app tracks current Bitcoin prices in 13 different currency values. The prices are retrieved from https://apiv2.bitcoinaverage.com/
The different currency values supported in the app are:
* AUD
* BRL
* CAD
* CNY
* EUR
* GBP
* HKD
* JPY
* PLN
* RUB
* SEK
* USD
* ZAR
